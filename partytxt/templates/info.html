GroupSMS

Group Chat without the apps. Text START to # to create a group.

Features

Group Chat via SMS

Create a group chat via SMS, No apps to download. Its Quick and Easy.

Party TxT lets group members choose how to interact with the group.
Commands include:
JOIN - joins the group
NAME - change your name
IGNORE - ignore all incoming messages
LEAVE - leave the group
WAKE - receive messages again

Custom Commands

Custom commands will be added in future version. Lets get this MVP off the ground first

Voice Messages

Create and listen to voice mail messages. If SMS is to hard to get your point across leave a detailed message

Live

All messages,images,and voice messages are available to your groups webpage.
